vimrc-in-gentoo
===============

my vimrc in gentoo

HOW TO USE
----------

    cp vimrc.local /etc/vim/
    cp .vimrc ~/

then install [vundle](https://github.com/gmarik/vundle)

    $ git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

and run vim, then install plugins:

    :BundleInstall
